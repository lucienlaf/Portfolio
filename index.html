01

Chifoumi

ISCB — Tours

Non disponible au public

Langages & Technologies

C# .NET

Demande du client

Développer une application de bureau ludique permettant à un utilisateur de jouer à Pierre-Feuille-Ciseaux contre l'ordinateur, avec des options de personnalisation (thème visuel, difficulté du bot).

Description du projet

Mini-jeu réalisé en C# sous WinForms. L'utilisateur choisit entre pierre, feuille ou ciseaux et s'affronte contre un bot. Le projet intègre un panel de personnalisation visuelle, un système de score, et différents niveaux de difficulté simulant des stratégies de l'ordinateur.

Ce projet n'est pas disponible au public.

02

Memory

ISCB — Tours

Non disponible au public

Langages & Technologies

C# .NET

Demande du client

Créer un jeu de mémoire configurable où le joueur peut choisir le nombre de paires de cartes à retrouver et le thème des images affichées.

Description du projet

Application de bureau réalisée en C# avec WinForms. Le joueur configure une partie en sélectionnant le nombre de paires et un thème d'images. Les cartes sont disposées aléatoirement sur la grille et le joueur doit les retourner par paires pour les identifier. Un compteur de coups et un chronomètre enrichissent l'expérience.

Demande du client

Dans le cadre du projet Cyber4All, développer un module de quiz interactif permettant de sensibiliser le grand public à la cybersécurité, avec différents thèmes et niveaux de difficulté.

Description du projet

Site web développé en PHP avec une base de données MySQL. Les résultats sont enregistrés et un tableau de scores encourage la compétition entre utilisateurs.
